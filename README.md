 🔍 Crime Pattern Analysis & Hotspot Detection using Machine Learning

**Mini Project | Python | Machine Learning | Data Science**

---

### 📌 Project Overview
This project analyzes crime data to:
- Detect **crime hotspots** using clustering (KMeans, DBSCAN)
- Identify **crime patterns** by time, location, and type
- Build a **crime prediction model** using classification (Random Forest)
- Visualize results with **heatmaps, plots, and interactive maps**

### 📦 Libraries Used
- `pandas`, `numpy` – Data handling
- `matplotlib`, `seaborn` – Visualization
- `sklearn` – Machine Learning
- `folium` – Interactive maps
- `scipy` – Statistical analysis
